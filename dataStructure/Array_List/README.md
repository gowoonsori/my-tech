
### 리스트
```
배열과 달리 원소들 간의 논리적인 순서로 연결되어 구성
```
###### :heavy_check_mark: 단순 연결 리스트

1. 리스트의 방향이 한쪽으로만 구성
1. 구현 방법 
  1. 한개의 노드에는 key값과 다음 노드의 주소를 가르킬 포인터 변수로 구성
1. 시간 복잡도 
  1. 노드의 개수가 n개일 경우 O(n)

###### :heavy_check_mark: 이중 연결 리스트

1. 리스트 방향이 양쪽으로 구성
1. 구현방법
  1. 한개의 노드에는 key값과 이전 노드와 다음 노드의 주소를 가르킬 포인터 변수 2개로 구성
  1. 첫 노드 생성시 left/right node pointer는 null로 초기화
  1. 노드 생성시 leftNode*는 왼쪽 노드를 rightnode*는 null로 생성후 원래 있던 끝 자락 노드의 right node*를 생성한 노드에 연결
