# KeyMap

나만의 KeyMap 기록

## 단어 찾기

- 페이지 내 같은 단어 : ctrl + d

- 한 단어 건너뛰고 다음 단어 : ctrl + q
- 단어 선택 취소 : ctrl + u

## 디렉토리 scope에서 파일 생성 | 페이지 scope에서 GENERATER 생성

- alt + n

## 디렉토리에서 scope에서 디렉토리 생성

- alt + d

## page내 scope에서 줄 삭제

- alt + d

## 현재 줄 아래로 복사

- ctrl + d

## 현재 줄 잘라내기

- ctrl + x

## 현재 줄 위아래 이동

- alt + ⬆⬇

## 함수 파라미터 값 확인

- ctrl + space

## 해당 커서의 실행가능 액션

- ctrl + N

## Generator,Getter/setter 생성

- Alt + N

## class test 파일 자동생성 및 자동 이동

- shift + alt + t

## 다음줄 엔터

- ctrl + enter

## 현재 변수의 반환 값 변수 자동 생성

- ctrl + alt + v

## Optimize import(사용하지 않는 import 제거)

- shift + alt + o

## 자동 정렬

- shift + alt + f

## 모든 페이지 내 검색

- shift + ctrl + f

## ssh(WSL2 terminal열기)

- ctrl + alt + s

## 메인 메소드 생성

- psvm

## System.out.println

sout
